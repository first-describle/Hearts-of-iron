💾 存档格式变化

旧存档 (当前)
{
  "name": "汉斯",
  "playerClass": 0,     ← 枚举int值
  "hp": 80,
  "currentScenario": 3,  ← 枚举int值
  "currentNodeId": "xx",
  "flags": ["saved_boy"],
  "unlockedScenarios": [0,1,2]
}
新存档 (改造后)
{
  "engineVersion": "2.0",
  "dlcId": "third_reich",
  "playerName": "汉斯",
  "playerClass": "infantry", ← 字符串ID
  "hp": 80,
  "morale": 65,
  "currentChapter": "stalingrad",
  "currentNodeId": "st_choice_5",
  "unlockedChapters": ["fall_gelb","britain","wolf_pack","stalingrad"],
  "flags": ["saved_boy", "chose_west"]
}

⚠️ 新存档不兼容旧存档。引擎启动时检测到旧格式会提示用户。旧版本存档 被标记为 legacy，不自动迁移（因为数据格式语义完全不同）。
